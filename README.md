# Facial-Recognition-Attendance-System

## Project Overview

Traditional attendance systems are manual and prone to human errors.  
This project automates attendance tracking using **facial recognition technology**.

The system performs the following workflow:

Camera Input → Face Detection → Feature Extraction → Face Encoding → ML Classification → Attendance Logging

It identifies individuals from a trained dataset and records their attendance automatically.

## Features

- Real-time face detection using **OpenCV**
- Face recognition using **face_recognition library**
- Machine learning classification using **Support Vector Machine (SVM)**
- Automated dataset generation using webcam
- Model training pipeline for face embeddings
- Flask-based web interface for system interaction
- Automated attendance marking
- Real-time identity recognition

## Applications

- Smart attendance systems in educational institutions
- Employee attendance tracking
- Access control systems
- Security surveillance
